ресурсы страницы распихнуть по папкам, в App.vue можно подгрузить css с помощью @import "./css/****.css";
он будет влиять на все подгружаемые элементы

можно импортировать css только в одну из страниц xxx.vue, или один из компонентов

как грузить js из файлов разберемся, но по идее легче писать локальный js для компонентов

mixins это скрипты, доступные везде внутри экспорта, импорт в main.js, потом app.mixin(название)
